## Google Chart Adapter Gallery
<script type="text/javascript" src="https://www.gstatic.com/charts/loader.js"></script>
<script type="text/javascript" src="../../../javascript/gallery1.js"></script>
    
The following charts show the output generated by the Morpheus Google charting adapter based on the examples presented in the 
[overview](overview). Note that this page takes a fairly long time to load as it imposes a fairly significant burden on the browser
to render this many charts in a single page. The Javascript that contains the charting data and instructions for these plots is code 
generated by the Morpheus adapter and can be viewed [here](../../javascript/gallery1.js).

Note how these charts are interactive in that they support click & drag to zoom and also support informative tooltips and other animations
when you mouse over data points or the legend in the chart.
